
- `pyserial`是必需的，用于串口监控功能
- `watchdog`用于文件变化自动编译，可选但推荐

## ⚙️ 手动操作命令

### 并行编译Make命令
```bash
# 在bootloader目录（推荐并行编译）
make clean          # 清理
make -j8           # 8线程并行编译（推荐）
make               # 单线程编译（不推荐）
make flash         # 烧录（如果配置了）

# 在application目录（推荐并行编译）
make clean          # 清理
make -j8           # 8线程并行编译（推荐）
make               # 单线程编译（不推荐）
make flash         # 烧录（如果配置了）

# 自动检测CPU核心数并行编译
make -j$(nproc)    # Linux
make -j%NUMBER_OF_PROCESSORS%  # Windows（需要环境变量）
```

### OpenOCD命令
```bash
# 烧录bootloader到内部Flash
openocd -f interface/stlink.cfg -f target/stm32h7x.cfg \
  -c "program bootloader/build/bootloader.hex verify reset exit"

# 烧录QSPI Flash
openocd -f interface/stlink.cfg -f target/stm32h7x.cfg \
  -c "init" \
  -c "reset halt" \
  -c "flash write_image application/build/application.bin 0x90010000" \
  -c "reset run" \
  -c "exit"

# 烧录元数据
openocd -f interface/stlink.cfg -f target/stm32h7x.cfg \
  -c "init" \
  -c "reset halt" \
  -c "flash write_image metadata_init.bin 0x90000000" \
  -c "reset run" \
  -c "exit"
```

### 文件操作
```bash
# 查看编译输出
ls -la bootloader/build/
ls -la application/build/

# 查看文件大小
du -h bootloader/build/bootloader.bin
du -h application/build/application.bin

# 检查二进制文件
hexdump -C application/build/application.bin | head
```

## 🚨 故障排除命令

### 编译问题
```bash
# 检查编译错误（并行编译）
python scripts/dev_debug.py build 2>&1 | grep error

# 强制重新编译（推荐并行）
cd bootloader && make clean && make -j8
cd application && make clean && make -j8

# 检查工具链
arm-none-eabi-gcc --version
make --version

# 检查CPU核心数
python -c "import multiprocessing; print(f'CPU核心数: {multiprocessing.cpu_count()}')"
```

### 烧录问题
```bash
# 检查调试器连接
openocd -f interface/stlink.cfg -f target/stm32h7x.cfg \
  -c "init" -c "targets" -c "exit"

# 检查Flash状态
openocd -f interface/stlink.cfg -f target/stm32h7x.cfg \
  -c "init" -c "flash info 0" -c "exit"

# 擦除Flash（慎用）
openocd -f interface/stlink.cfg -f target/stm32h7x.cfg \
  -c "init" -c "reset halt" -c "flash erase_sector 0 0 last" -c "exit"
```

### 运行时问题
```bash
# 串口监控查看启动日志
python scripts/dev_debug.py monitor COM3

# 检查文件完整性
md5sum bootloader/build/bootloader.bin

```
